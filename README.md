# Бот «Матрицы Сокальской»

## Быстрый запуск

### 1. Установить Python 3.10+

### 2. Установить зависимости
```bash
pip install python-telegram-bot==20.7
```

### 3. Запустить бота
```bash
python bot.py
```

Всё — бот работает. Напишите ему `/start` в Telegram.

---

## Запуск на бесплатном сервере (Railway)

1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. Создайте новый проект → «Deploy from GitHub repo»
3. Загрузите папку с файлами на GitHub (или используйте Railway CLI)
4. В Railway добавьте переменную окружения:
   - `BOT_TOKEN` = ваш токен
5. В `bot.py` замените строку с токеном на:
   ```python
   import os
   TOKEN = os.environ["BOT_TOKEN"]
   ```
6. Railway автоматически запустит бота

## Команды бота

- `/start` — начать тест заново

## Структура теста

- 20 вопросов с перемешанными вариантами ответов
- Мультиселект — можно выбрать несколько вариантов
- Ранжирование выбранных ответов кнопками ▲ ▼
- Взвешенный подсчёт: первый выбранный вариант важнее последнего
- Результат: матрица, диаграмма, рекомендации, ловушки
